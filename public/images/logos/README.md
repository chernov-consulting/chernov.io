# Company Logos

**Note**: Simple SVG text placeholders have been created for now. Please replace them with official logos from the sources below.

## How to Replace Placeholders:

1. Download the official logo from the company's press/media page (links below)
2. Save it in this directory with the exact filename listed
3. You can use PNG, SVG, or other image formats
4. The website will automatically use your new logos

---

Please download the official logos from the following sources and save them in this directory:

## Required Logo Files:

### 1. Amazon (`amazon-logo.svg` or `.png`)
- Visit: https://press.aboutamazon.com/
- Download the Amazon logo (preferably horizontal version)
- Save as: `amazon-logo.svg` or `amazon-logo.png`
- Format: SVG or PNG with transparent background
- Recommended: Use the black/dark version

## Logo Specifications:
- **Format**: PNG with transparent background
- **Color**: Dark/black versions work best (will be grayscaled initially)
- **Size**: At least 500px width for quality
- **Aspect**: Horizontal/landscape orientation preferred

## Note:
The logos will be displayed with:
- Grayscale filter by default (60% opacity)
- Full color on hover
- Maximum display size: 120px wide × 50px tall
